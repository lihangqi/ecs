# AvailableResourceCreationType {#AvailableResourceCreationType .reference}

允许创建的资源类型。

## 节点名 {#section_u2s_ffp_ydb .section}

接口决定。

## 子节点 {#ResponseParameter .section}

|名称|类型|描述|
|:-|:-|:-|
|ResourceTypes|String|资源类型，如下：-   Instance：支持实例创建
-   IoOptimized：支持IO优化的实例创建
-   Disk：支持磁盘创建
-   VSwitch：支持专有网络创建

|
